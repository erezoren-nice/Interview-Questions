### 1. [Simple Variable Calculator](src/main/java/interview/questions/backend/calculator/REDME.md)
**Difficulty:** Mid-Senior Backend

Implement an interactive calculator that maintains variables in memory and supports assignment operations with pre-increment and post-increment operators.

**Key Concepts:** Variable management, operators, command parsing, interactive CLI

---

### 2. [Excel Sheet Calculator](src/main/java/interview/questions/backend/excelsheet/REDME.md)
**Difficulty:** Mid-Senior Backend

Build an Excel-like spreadsheet calculator that can store cell values and evaluate formulas with cell references.

**Key Concepts:** Formula evaluation, dependency resolution, cycle detection, memoization

---

### 3. [Delayed Job Scheduler](src/main/java/interview/questions/backend/jobscheduler/REDME.md)
**Difficulty:** Mid-Senior Backend

Create a job scheduler that executes tasks after specified delays in correct chronological order.

**Key Concepts:** Priority queues, delayed execution, timing, concurrent programming

---

### 4. [LRU Cache Implementation](src/main/java/interview/questions/backend/lru/REDME.md)
**Difficulty:** Mid-Senior Backend

Implement a Least Recently Used (LRU) cache with O(1) get and put operations.

**Key Concepts:** Data structures, doubly linked list, hash maps, cache eviction policies

---

### 5. [Retryable Event Processor](src/main/java/interview/questions/backend/retryableevent/REDME.md)
**Difficulty:** Mid-Senior Backend

Build an event processor that executes events asynchronously with exponential backoff retry logic.

**Key Concepts:** Asynchronous processing, retry patterns, backoff strategies, thread pools
